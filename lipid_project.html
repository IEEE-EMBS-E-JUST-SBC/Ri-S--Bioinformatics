Precision Lipid Therapy Prediction

Target Venue: IEEE Conference 2026  ·  Summer Research Timeline

Project Overview

Lipid-lowering therapies like ANGPTL3 inhibitors show dramatic clinical benefits, but identifying who will respond is complex. This project fuses Polygenic Risk Scores (PRS) derived from GWAS summary statistics with observed phenotypic lipid trajectories from NHANES to build an in-silico precision medicine classifier. The goal is to stratify patients for personalized therapy prior to trial enrollment.

Command Line (PLINK/PRSice-2)

GWAS → PRS Construction

Parse GLGC GWAS summary statistics for TG and LDL traits. Apply LD pruning (r² < 0.1, window 250kb) to select independent SNPs. PRSice-2 computes per-patient weighted sum scores — the genetic predisposition to high TG/LDL.

# LD pruning in PLINK
plink --bfile cohort_genotypes \
--indep-pairwise 250 50 0.1 \
--out ld_pruned

# PRSice-2 score computation
PRSice --base GLGC_TG.sumstats \
--target cohort_genotypes \
--snp SNP --A1 A1 --stat BETA \
--out TG_PRS
Python Post-Processing

Score Standardisation

Raw PRS values are standardised to Z-scores across the cohort (mean 0, SD 1). Z-scores enable clinical interpretation: a patient at PRS Z=+2.5 is at the 99th percentile for genetic TG liability — a strong prior for drug response.

import pandas as pd
from scipy import stats

prs = pd.read_csv('TG_PRS.best')
prs['TG_PRS_Z'] = stats.zscore(prs['PRS'])
prs['LDL_PRS_Z'] = stats.zscore(prs['LDL_PRS'])

XGBoost Classifier

Gradient-Boosted Ensemble

XGBoost with tree-based ensembles handles the heterogeneous feature types (continuous genomic Z-scores, binary statin flag, log-transformed lab values) without manual scaling. Bayesian hyperparameter search (Optuna) optimises depth, learning rate, and subsample ratio.

Evaluation Strategy

Cross-Validation Setup

Stratified 5-fold CV with group-aware splitting (patients from the same household in the same fold). Primary metric: AUROC. Secondary: precision-recall AUC (important under class imbalance where non-responders outnumber responders ~3:1).

from sklearn.metrics import roc_auc_score

Probability Calibration

Isotonic Regression

Raw XGBoost output probabilities are often overconfident. Isotonic regression (fit on a held-out calibration set) maps model scores to true posterior probabilities, ensuring a predicted 70% score genuinely corresponds to 70% of patients responding — critical for clinical decision-making.

from sklearn.calibration import CalibratedClassifierCV

calibrated = CalibratedClassifierCV(
base_xgb, method='isotonic', cv='prefit')
calibrated.fit(X_cal, y_cal)

Explainability

SHAP TreeExplainer

TreeExplainer computes exact SHAP values for XGBoost in linear time. Summary plots reveal the global feature importance ranking; force plots explain individual predictions — showing which specific PRS and clinical values pushed a patient toward or away from the responder classification.

shap_vals = explainer.shap_values(X_test)

# Beeswarm summary plot
