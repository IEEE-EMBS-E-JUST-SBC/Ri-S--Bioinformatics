Viral Mutation Hotspot Prediction

 ·  Summer Research Timeline

Project Overview

Viral mutation hotspots are genomic loci under high evolutionary pressure. Identifying these regions before new variants emerge enables vaccine designers to prioritize epitope targets. This track builds machine learning models that predict high-risk mutation hotspots using an integrated view of structural conservation, evolutionary signals, and deep protein language model embeddings from ESM-2.

Alignment

Codon-Aware MSA (MAFFT)

Execute codon-aware Multiple Sequence Alignment yielding ~9,600 aligned codon positions. Codon-awareness prevents frameshifts that corrupt dN/dS calculations downstream.

# MAFFT codon-aware alignment
mafft --auto --thread 8 \
--codon genomes.fasta \
> aligned_codons.fasta

# QC: mask positions >50% gap
trimal -gt 0.5 -in aligned.fasta
Structural Features

ConSurf + Entropy Scores

Compute per-residue conservation scores (ConSurf), Shannon entropy over the column distribution, relative solvent accessibility (RSA) from AlphaFold structures, and PSIPRED secondary structure predictions.

import numpy as np
def shannon_entropy(col):
freqs = np.unique(col, return_counts=True)[1]
p = freqs / freqs.sum()
return -np.sum(p * np.log2(p + 1e-9))
Evolutionary + Embeddings

dN/dS + ESM-2

Sliding-window dN/dS ratios (KaKs_Calculator), CAI, GC content. ESM-2 (650M params) extracts 1280-dim vectors from the final hidden layer per codon position — providing learned evolutionary context unavailable from sequence alone.

from esm import pretrained
model, alphabet = pretrained.esm2_t33_650M_UR50D()
batch = alphabet.get_batch_converter()
_, _, tokens = batch(sequences)
results = model(tokens, repr_layers=[33])
embeddings = results["representations"][33]
Architecture

1D CNN-LSTM Network

Input: concatenated 1295-dim feature matrix per codon. Three parallel 1D Conv layers (kernels 3, 5, 7) capture local motifs at different scales; their outputs are concatenated and fed to an LSTM (128 hidden units) for long-range genomic dependencies. Output: hotspot probability per codon.

# Parallel convolution branches
x3 = Conv1D(64, kernel_size=3, activation='relu')(x)
x5 = Conv1D(64, kernel_size=5, activation='relu')(x)
x7 = Conv1D(64, kernel_size=7, activation='relu')(x)
merged = Concatenate()([x3, x5, x7])
out = LSTM(128, return_sequences=True)(merged)
Training Strategy

Class Imbalance & Calibration

Hotspot labels are rare (<5% of codons). SMOTE oversampling addresses imbalance in training. Weighted binary cross-entropy during training. Threshold tuned to maximize sensitivity at 95% specificity — the clinically relevant operating point.

Benchmarking

Validation Against Known VoCs

Cross-validate predictions against catalogued SARS-CoV-2 variants of concern. Hotspot predictions should overlap with known immune-escape and ACE2-binding residues on the Spike protein — providing biological ground truth for model evaluation beyond held-out test accuracy.
